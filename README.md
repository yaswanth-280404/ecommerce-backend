A **simple yet professional E-commerce backend** built using **Spring Boot**, **Spring Data JPA**, and **MySQL**.  
The application supports user management, product categorization, order placement, and API documentation using Swagger.

---

## Features

- User creation and retrieval
- Category and product management
- Order placement and order history
- One-to-Many and Many-to-One JPA relationships
- Cascade operations for orders and order items
- Global exception handling
- Swagger / OpenAPI documentation

---

## Tech Stack

- **Java 21**
- **Spring Boot 3.2.x**
- **Spring Data JPA (Hibernate)**
- **MySQL**
- **Maven**
- **Swagger (Springdoc OpenAPI)**

---

## Prerequisites

- Java 21
- Maven
- MySQL
